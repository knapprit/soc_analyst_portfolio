## Подозрительная активность в Microsoft 365 (Search → MailItemsAccessed → Set-Mailbox SYSTEM)
На Снимок экрана (20).png зафиксирована последовательность событий Microsoft 365, характерная для **компрометации почтового ящика**:

1. **Поиск по почтовому содержимому (`Search`)**  
2. **Доступ к письмам (`MailItemsAccessed`)**  
3. **Изменение конфигурации ящика (`Set-Mailbox`) от имени `NT AUTHORITY\SYSTEM`**

Такая цепочка является **подозрительной**, поскольку объединяет разведку, доступ к данным и административные изменения в коротком промежутке времени.
### Зафиксированные типы событий
- `Set-Mailbox` (исполнитель — `NT AUTHORITY\SYSTEM`)
- `Search`
- множественные `MailItemsAccessed`
- сервисные и пользовательские события входа

  - `Search` часто используется злоумышленниками для быстрого поиска ценной переписки  
- `MailItemsAccessed` подтверждает **фактический доступ к письмам**, а не просто просмотр метаданных  
- `Set-Mailbox` под SYSTEM может использоваться для:
  - закрепления доступа (persistence)
  - ослабления аудита
  - перенаправления почты
- События идут **последовательно**, что указывает на осмысленное поведение, а не фоновый шум
